This is a [Next.js](https://nextjs.org/) TypeScript + Sass template.

## Getting Started

To import this template:

```bash
npx create-next-app ts-sass-next -e https://github.com/Alteras1/Next.js-TypeScript-Sass
```

## Versions

- Next.js - 10.0.3
- TypeScript - 4.1.3
- Sass - 1.30.0
